What is Windows Longhorn?

"Windows Longhorn" was the codename for the development of Windows Vista. It was intended to be a version between XP and Vista. Development started in May 2001 — even before the release of Windows XP. Longhorn was never officially released as a standalone Windows version, and a few years later, Windows Vista was launched.

Okay. So what is Windows Longhorn Ultimate?

Windows Longhorn Ultimate is an unofficial standalone version of Windows Longhorn founded by two (kind of unocuppied high schoolers) and later getting supported by friends and even teachers!

Why are you doing this?

Why?? Because we LOVE Windows Longhorn! It's our favorite version of Windows. We want to make a great experience for all users, and even for your bussiness! We feel that it's unfair a great Windows version has to be buggy and hard to use.
